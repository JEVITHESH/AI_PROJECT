# Chemisafe Predictor Pro - Project Report

## 1. Executive Summary
**Chemisafe Predictor Pro** is an advanced AI-powered application designed to predict chemical reaction outcomes and safety hazards in real-time. By leveraging Large Language Models (LLMs) via the Groq API, the system analyzes chemical combinations to identify potential dangers, providing distinct safety scores, hazard levels (Safe, Caution, Danger, Lethal), and intelligent recommendations for safe handling.

## 4. Implementation Details

### 4.1 Backend Processing (`server.py`)
The backend exposes a single endpoint `POST /predict`.
1.  **Input**: Receives a list of chemical names.
2.  **Prompt Engineering**: Constructs a specialized system prompt enforcing a strict JSON output schema.
3.  **AI Query**: Sends the request to Groq SDK with `temperature=0.1` for deterministic, reliable safety outputs.
4.  **Validation**: Parses and validates the JSON response to ensure it meets the `PredictionResult` schema before sending it to the frontend.

### 4.2 Frontend Logic (`App.tsx` & `geminiService.ts`)
- **API Integration**: `geminiService.ts` handles the `fetch` calls to the local FastAPI server.
- **Error Handling**: Graceful degradation if the server is offline or the AI returns invalid data.
- **Local Storage**: Uses browser `localStorage` to persist user sessions and history without a database requirement for the MVP.

2.  **Frontend Setup**:
    ```bash
    npm install
    npm run dev
    ```
